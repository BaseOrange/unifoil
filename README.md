# What is this?

UniFoil（单向箔）是一个轻量级、跨平台、单文件运行的内网单向文件收集工具。

# Tauri + Vue 3

这个模板应该可以帮助你开始在Vite中使用Tauri + Vue 3进行开发。该模板使用Vue 3 `<script setup>` SFCs， 查看 [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) 了解更多信息.
